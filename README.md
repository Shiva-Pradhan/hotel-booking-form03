# Hotel Booking Website 🏨
🏨 LuxuryStay – Modern Hotel Booking Website

LuxuryStay is a clean and responsive hotel booking website designed to provide users with a smooth and user-friendly booking experience. The project allows users to explore room options, view amenities, select stay dates, calculate booking costs dynamically, upload a government ID, and complete a booking through an interactive interface.

## 🌟 Project Overview
This project simulates a real-world hotel booking system using front-end web technologies. It focuses on form validation, date-based stay calculation, occupancy rules, and a complete payment flow without using any backend server or database.
The entire booking process runs in the browser and ends with a booking confirmation screen displaying all booking details.


## 🛠️ Technologies Used
- **HTML5** – Structure of the website  
- **CSS3** – Styling and responsive layout  
- **JavaScript (Vanilla JS)** – Booking logic, validations, and dynamic price calculation  


## ✨ Features
- Responsive hotel booking interface  
- Room selection with dynamic pricing  
- Date-based stay calculation (check-in & check-out)  
- Guest and occupancy validation  
- Government ID upload validation (PDF / JPG / PNG)  
- Live price preview before booking  
- Payment method selection  
- Booking confirmation page with unique booking ID  


## 🏠 Rooms Available

Each room includes pricing, occupancy limits, and listed amenities.
